## Setup
```sh
// Node - node 8 is required
$ npm install
$ npm start
```

`npm start` runs `webpack --watch` and `node api.js` for you.

## UI

The basic html page in `public/index.html` renders at `localhost:8080`.

Two URL are accessible in the browser:

'.../' & '.../settings/profile';
